# Hi, I'm Sachini Weligamage

**Data Engineer | BI & Data Analytics Engineer**


## About Me

I'm a Data Engineer with 5+ years of experience designing and optimizing cloud based data
platforms, specializing in **Azure Databricks**, **Medallion architecture**, and scalable
**Spark & Kafka** pipelines. I have full cycle experience across the data lifecycle from
**data ingestion** and **data cleaning** to **data testing/validation**, through to delivering
**analytics via reports and dashboards** for business stakeholders. I focus on building
reliable, cost efficient data systems that turn raw data into trustworthy, decision ready insights.

* End to end ETL/ELT pipeline design (ingestion, transformation, loading, orchestration)

**Data Quality & Reliability Engineering**
* End to end data validation and testing across source & target systems
* Data quality checks across Bronze/Silver layers to ensure accuracy and consistency

**Cloud, Security & DevOps**

* Cloud based data platform design and administration
* CI/CD for data pipelines, Git based version control

**Programming, Databases & Analytics**
